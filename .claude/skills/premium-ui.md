# Premium UI System

Design philosophy:
- Minimal, elegant, premium
- No generic AI layouts
- Strong visual hierarchy
- Large typography
- Generous whitespace
- Grid-based structure

Rules:
- Max width: 1200px
- Use spacing rhythm consistently
- Avoid clutter
- Each section must have a single purpose

Visual direction:
- Luxury SaaS / fintech aesthetic
- Neutral or dark backgrounds
- One refined accent color only
- Soft shadows
- Rounded corners (8px–16px)

Avoid:
- template feel
- overused gradients
- crowded sections
- random colors
- too many cards

The UI should feel expensive, intentional, and modern.
