# GSAP Motion Principles

Animations must feel:
- smooth
- premium
- intentional
- cinematic

Use:
- GSAP + ScrollTrigger
- fade + translateY reveals
- staggered timing
- easing functions
- smooth sequencing

Rules:
- Animations should guide attention
- Never animate everything at once
- Use delays between elements
- Keep transitions smooth (0.6s–1s)

Preferred effects:
- subtle parallax
- mask reveals
- staggered text
- progressive section reveals
- smooth hover elevation

Avoid:
- flashy animations
- excessive scaling
- fast aggressive movement
- distracting effects

Target feel:
Awwwards-quality motion with restraint.
