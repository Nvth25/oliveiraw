# Layout & Composition System

Use a structured layout system.

Rules:
- Sections must breathe
- Large vertical spacing
- Balanced composition
- Clear alignment

Preferred layouts:
- split sections
- asymmetrical balance
- editorial-inspired layouts
- bento grid where appropriate

Use:
- max 2–3 columns
- visual rhythm
- progressive storytelling

Avoid:
- long unbroken sections
- visual chaos
- repetitive card grids

Goal:
The layout should feel curated and premium.
