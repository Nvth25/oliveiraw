# Three.js Visual Direction

Three.js should enhance the brand experience, not dominate it.

Preferred visuals:
- abstract geometry
- particles
- fluid movement
- depth-based motion
- subtle lighting

Rules:
- lightweight
- elegant
- minimal
- performant

Use primarily in:
- hero sections
- background layers
- transitions

Avoid:
- gaming aesthetics
- excessive effects
- visual overload

Goal:
Create a modern premium digital atmosphere.
