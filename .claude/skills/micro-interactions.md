# Micro-Interaction System

Interactions should feel subtle and premium.

Use:
- hover scale (1.02–1.05)
- smooth transitions
- soft shadow changes
- button feedback
- cursor responsiveness

Animation timing:
- 0.2s–0.4s
- ease-in-out

Goal:
Make the interface feel alive without distraction.

Avoid:
- gimmicky cursor effects
- exaggerated motion
- excessive interaction noise
