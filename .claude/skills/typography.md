# Typography Standards

Typography is the primary design element.

Rules:
- Strong headline hierarchy
- Large hero titles
- Clean sans-serif fonts
- High contrast between headings and body text

Spacing:
- generous line-height
- balanced letter spacing
- clear paragraph rhythm

Preferred style:
- bold but elegant
- modern fintech aesthetic
- minimal but impactful

Avoid:
- tiny body text
- inconsistent font sizes
- crowded typography
- more than 2 fonts

Typography should create confidence and clarity.
